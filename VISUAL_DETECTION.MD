# Visual Direction

Visual identity:

- black / deep navy backgrounds
- subtle violet accent
- cinematic gradients
- restrained motion
- technical minimalism

Avoid:

- colorful dashboards
- neon overload
- playful illustrations
- corporate stock imagery

Preferred visuals:

- desktop screenshots
- session logs
- structured tables
- timeline interfaces
- code-like UI
- operational control surfaces

Motion:

- slow
- weighty
- elegant

Design feeling:
"Secure infrastructure software."
