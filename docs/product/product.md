# Access by AMS

Access by AMS is a desktop-first assessment integrity platform built for serious online evaluations.

It combines:

- fullscreen controlled sessions
- candidate activity logging
- reviewer timelines
- written response workflows
- lightweight assessment infrastructure

The platform is designed for:

- hiring rounds
- olympiads
- university evaluations
- scholarship tests
- high-trust written assessments

Access by AMS is NOT:

- a generic LMS
- a classroom management tool
- a casual quiz platform
- a mass-market exam portal

The product should feel:

- premium
- technical
- minimal
- trustworthy
- operationally serious

Core philosophy:
"Capture assessment intent with clarity."
