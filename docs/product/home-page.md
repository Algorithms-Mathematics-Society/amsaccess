# Landing Page Rules

The homepage is NOT an assessment creation UI.

It is:

- a product showcase
- software landing page
- desktop app presentation
- infrastructure product site

Prioritize:

- screenshots
- download CTA
- pricing CTA
- feature explanation
- trust positioning

De-prioritize:

- inline assessment builders
- long forms
- workflow diagrams
- marketing clutter

The homepage should feel:

- premium
- sparse
- cinematic
- operational
