# Copy Guidelines

Tone:

- concise
- premium
- technical
- calm
- confident

Avoid:

- startup buzzwords
- hype language
- flashy SaaS copy
- aggressive anti-cheat wording

Preferred style:
Short declarative statements.

Examples:

- "Run controlled written rounds."
- "Review candidate timelines clearly."
- "Capture session evidence automatically."
- "Built for high-trust evaluations."

Avoid:

- "Revolutionary"
- "Next-generation"
- "AI-powered disruption"
- "Cheat-proof"

The product should sound:

- infrastructural
- operational
- trustworthy
