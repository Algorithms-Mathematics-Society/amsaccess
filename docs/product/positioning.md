# Positioning

Access by AMS should be positioned as:

"Assessment infrastructure for serious online rounds."

NOT:

- quiz software
- Google Forms replacement
- coaching platform
- anti-cheat spyware

The value proposition is:

- controlled assessment environments
- reviewer-ready evidence
- operational clarity
- structured candidate workflows

Messaging themes:

- integrity
- structure
- trust
- operational confidence
- reviewability

Avoid exaggerated claims:

- "prevents cheating"
- "impossible to bypass"
- "AI-powered surveillance"

Preferred language:

- fullscreen session
- activity evidence
- reviewer workflow
- candidate session timeline
- desktop assessment shell
- integrity review
